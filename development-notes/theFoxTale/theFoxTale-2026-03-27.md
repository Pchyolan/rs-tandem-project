- **Создала единую фабрику виджетов** (`WidgetFactory`), которая по типу виджета - quiz, memory-game - создаёт нужный экземпляр. Теперь `TicketPageController` может работать с любыми виджетами без привязки к конкретному классу и выглядит как билет.

    - функциональный модуль (`quiz-widget-functional.ts`) – формирует всю DOM-логику без классов, используя только функции.
    - Аналогично поступила с домашней страницей (`home-page-functional.ts` + `home-page.ts`).

- **Добавила универсальный тип `WidgetContext`** – теперь все виджеты получают одинаковый набор колбэков (`onReady`, `onComplete`, `widgetId`).

- **Обновила страницы:**

- Для `TicketPageController` ввела массив `items: TicketItem[]`, где каждый элемент содержит `type` и `id`. Теперь при переходе по стрелкам или завершении виджета контроллер просто создаёт новый виджет через фабрику, не заботясь о его типе.
